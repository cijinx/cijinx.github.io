# BunJS文档

>"Bun" 是用于 JavaScript 和 TypeScript 应用程序的一站式工具包。它以一个名为 bun 的单一可执行文件的形式发布。
>
>在其核心是 Bun 运行时，它是一个快速的 JavaScript 运行时，设计成可替代 Node.js。它是用 Zig 编写的，并在底层由 JavaScriptCore 驱动，大大减少了启动时间和内存使用。

## 安装

::: code-group
```sh [Homebrew]
brew install oven-sh/bun/bun
```
```sh [npm]
npm install -g bun
```
:::
